Les applications mobiles de DroneLogbook

La solution DroneLogbook est basée sur une infrastructure cloud avec un stockage sécurisé des données, une application web en ligne accompagnée de notre application mobile pour un accès hors ligne et une synchronisation facile avec votre compte basé sur le cloud. De plus, l'application mobile vous permet d'accéder aux dernières mises à jour de l'état de l'espace aérien, aux conditions météorologiques locales et à l'indice solaire qui peuvent affecter vos opérations.


Une autre application mobile Android, DLBSync, simplifie l'importation de vos vols à partir des principales applications de contrôle de vol de drone directement dans votre compte DroneLogbook. Cette application peut synchroniser les vols dans DLB Sync à partir de vos applications de contrôle de vol lorsque vous êtes hors ligne ou en cas de mauvaise couverture mobile, puis télécharger les vols vers votre compte DroneLogbook lorsque vous avez une couverture mobile ou WIFI.
