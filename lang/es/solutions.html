Soluciones de DroneLogbook

DroneLogbook está trabajando activamente para proporcionar soluciones específicas para la industria que ayuden a las empresas a gestionar mejor sus operaciones con drones. Reconocemos que diferentes industrias tienen diferentes formas de operar, y como proveedor de soluciones de software para la industria de drones, debemos ofrecer soluciones diferentes para satisfacer las diversas necesidades de educación, seguridad pública, energía, construcción, comercio general y otros sectores. Por lo tanto, hemos creado plataformas especialmente diseñadas para adaptarse a las necesidades de estos sectores industriales.

DroneLogbook también ofrece una versión de etiqueta privada (White Label) de nuestra plataforma para aquellos con operaciones con drones más complejas y/o a mayor escala. La plataforma es modular y configurable, con un panel de administración detallado que proporciona notificaciones personalizadas y una supervisión operativa detallada.

DroneLogbook para Comercial
DroneLogbook fue creado para ayudar a los operadores comerciales de UAV a planificar, gestionar y hacer un seguimiento de sus operaciones de vuelo con drones para mantener el cumplimiento normativo y gestionar mejor su negocio. Nuestra plataforma proporciona todas las herramientas necesarias para respaldar las operaciones comerciales con drones. A medida que la industria crece, seguimos añadiendo nuevas características e integrándonos con socios para proporcionar una solución integral y sin problemas para la gestión de flotas y el cumplimiento normativo de extremo a extremo.
DroneLogbook para Seguridad Pública
www.safetydrone.org
Esta versión de DroneLogbook está diseñada para satisfacer las necesidades técnicas y el lenguaje del sector de seguridad pública. Además, también tiene un precio que refleja los patrones de uso y los requisitos presupuestarios de los departamentos de policía, bomberos, servicios médicos de emergencia y planificación de emergencias. Nuestro único requisito para utilizar este sitio es que debes usar tu correo electrónico oficial de trabajo para registrarte, ya que está destinado exclusivamente a los primeros respondedores.
DroneLogbook para Educación
Si eres una institución educativa o de entrenamiento de drones y necesitas un plan que te permita rotar rápidamente a los estudiantes, contáctanos sobre nuestra opción para el sector educativo. Estos planes son una versión de nuestra plataforma de etiqueta privada diseñada para permitirte reclutar estudiantes, capacitarlos, hacer un seguimiento de sus vuelos y luego rotarlos, todo dentro de un presupuesto fijo. Este programa es una excelente solución para cursos de entrenamiento de drones/rpas/uav públicos y privados, desde universidades de cuatro años hasta programas de capacitación sindical.
DroneLogbook para Entrega
(Part 135 / Part 91)
DroneLogbook se está involucrando cada vez más en la entrega y operaciones BVLOS, ofreciendo soluciones personalizadas para empresas interesadas en estas áreas. Para estos clientes, ofrecemos soluciones de desarrollo personalizadas para gestionar flujos de trabajo operativos específicos, necesidades de mantenimiento e inspección, y requisitos de informes. Nuestro equipo de desarrollo trabaja en conjunto con el equipo de operaciones del cliente para entregar soluciones configurables y escalables.
DroneLogbook para Agricultura
www.dronelogbook.ag
DLB Ag Edition fue desarrollado teniendo en cuenta al operador UAV de la FAA Parte 137 y combina la potencia de DLB con las necesidades del aplicador agrícola personalizado. DLB Ag Edition fue creado con un flujo de trabajo específico para garantizar un intercambio preciso de información de aplicación desde el productor hasta el piloto remoto.
DroneLogbook para Fabricantes
Si eres un fabricante de drones, DroneLogbook puede proporcionarte una plataforma de etiqueta privada para ayudarte a brindar un mejor servicio a tus clientes. Con nuestra oferta personalizada, puedes ofrecer a tus clientes una solución integral de gestión de flota y registro de drones. Esta plataforma te permite interactuar con tus clientes durante todo el ciclo de vida del producto, asegurando una excelente experiencia para el cliente y proporcionándote información valiosa sobre el rendimiento de tus drones.
Versión de etiqueta privada (etiqueta blanca)
La versión de etiqueta privada (White Label) de DroneLogbook es una plataforma modular y configurable, una instancia en la nube dedicada de nuestro software público de cumplimiento de operaciones con drones y gestión de flotas de drones.
Las características clave de esta plataforma en comparación con el sitio público Dronelogbook.com son:

Estructura de datos:
  • - Tablas de datos personalizables (Tipos de vuelo, Equipos, Componentes de drones)
  • Formularios y plantillas personalizables:
        - Drone Inspection
        - Mission Plan
        - Custom Reports (Access All Data Tables)
        - Custom Documents – Checklists, Form Builder, Risk Assessment

    • Operaciones:
  • - Capacidad para separar a los usuarios en organizaciones separadas con supervisión principal. También se puede compartir información entre organizaciones si es necesario.
  • - Solicitud de misión: capacidad para que terceros delimiten áreas de vuelo (manteniendo sus datos ocultos)
  • - Notificación automática si se activan eventos (vuelo, operaciones, cumplimiento, personal). Cosas como la altitud de vuelo, inspecciones atrasadas, vigencia de pilotos, etc.
  • - Plataforma configurable que permite a los usuarios mantener sus flujos de trabajo


    • Administrativo:
  • Customer branding (colors/logo)
  • Private cloud storage infrastructure. On-demand specific region/country hosting available
  • Customizable (Custom programming available to modify platform for a fee)
  • Less expensive than our public server at scale


    • Esta plataforma es muy atractiva para organizaciones con múltiples oficinas/líneas de negocio que necesitan gestionar operaciones en crecimiento. Además, ofrecemos una versión especial para Seguridad Pública que se modifica para adaptarse a sus necesidades. Hay precios especiales disponibles para organizaciones de Seguridad Pública, Gobierno y Educación.

    Sectores que actualmente utilizan etiquetas privadas:
    Educación, Energía, Gobierno, Minería, Medios de comunicación, Seguridad Pública, Agricultura, Propietarios de activos (Transporte, Telecomunicaciones).

    Aplicaciones móviles de DroneLogbook

    La solución de DroneLogbook se basa en una infraestructura en la nube con almacenamiento seguro de datos, una aplicación web en línea acompañada de nuestra aplicación móvil para acceder sin conexión y sincronizar fácilmente con tu cuenta basada en la nube. Además, la aplicación móvil te proporciona acceso a las últimas actualizaciones del estado del espacio aéreo, el clima local y la lectura del índice solar que podrían afectar tus operaciones.


    Otra aplicación móvil para Android, DLBSync, simplifica la importación de tus vuelos desde las principales aplicaciones de control de vuelo de drones móviles directamente a tu cuenta de DroneLogbook. Esta aplicación puede sincronizar vuelos en DLB Sync desde tus aplicaciones de control de vuelo cuando estás sin conexión o con una mala cobertura móvil, luego cargar los vuelos en tu cuenta de DroneLogbook cuando tengas cobertura móvil o WIFI.
